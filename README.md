# yaohata
Framexs上で動作するテーマ

## 解説
Framexs上で動作するテーマの参考用プロダクトです。
テーマはXML、FTML、CSS、JSのひと塊です。

## 使い方
スタティックサイトでは<?framexs.skeleton yaohata/main.ftml?>で

## デモ
npm run server

http://localhost/index.xhtml

## 環境
最新のブラウザとFramexs1.20以降に対応します。

## インストール
このリポジトリをGitでクローンするかリリースからダウンロードします。
